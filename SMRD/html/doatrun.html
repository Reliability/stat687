doatrun {SMRD}R Documentation

#####

Description

data truncated because of burn-in and removal of an unknown number of DOAs

Format

A data.frame with 48 rows and 6 variables:

[, 1] lower Start of an inspection interval Numeric
[, 2] upper End of an inspection interval Numeric
[, 3] censor Event observed in the interval (right-censored/interval-censored) Categoric
[, 4] count Number of events observed in the interval Numeric
[, 5] truntime Truncation time Numeric
[, 6] truntype Truncation type Categoric

Source

Berkson, J. (1966), Examination of randomness of alpha-particle emissions, in Festschrift for J. Neyman, Research Papers in Statistics, F. N. David, Editor, New York: John Wiley & Sons
