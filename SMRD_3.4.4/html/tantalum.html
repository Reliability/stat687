tantalum {SMRD}R Documentation

Electrolytic capacitor accelerated life test data

Description

data truncated because of burn-in and removal of an unknown number of DOAs

Format

A data.frame with 48 rows and 5 variables:

[, 1] hours Accumulated time at event Numeric
[, 2] event Event observed at hours (failure/right-censored) Categoric
[, 3] count Number of events observed at hours Numeric
[, 4] volts Voltage applied to the unit Categoric
[, 5] celsius Temperature applied to the unit Categoric

Source

Singpurwalla, Castellino, and Goldschen (1975)
